# Bboggl 프로젝트 브리핑 (Claude.ai 채팅에서 인수인계)

이 문서는 Claude.ai 채팅에서 Bboggl 프로젝트를 기획·프론트엔드 프로토타입 제작하며
결정된 내용을 정리한 것입니다. Claude Code 세션 시작 시 이 파일을 함께 읽어주세요.

## 제품 개요
일정을 두서없이 텍스트로 나열하면 자동으로 캘린더에 정리해주는 서비스.
캘린더 정리를 귀찮아하거나 어려워하는 사람들을 위한 서비스가 목표.

## 디자인 시스템 (변경 없이 유지)
- 색상: Aubergine #4A154B(포인트) / 액션 블루 #1264A3 / Green #2EB67D / Yellow #ECB22E / Text #1D1C1D / Background #FFFFFF
- 버튼: border-radius 10px, 라운드형
- 폰트: Poppins (템플릿 적용 시 Playfair Display / Archivo Black / Baloo 2 추가 사용)
- 아이콘: 둥근 라인 스타일 (stroke 기반)

## 완성된 화면 (프론트엔드)
- **LandingPage**: 텍스트→캘린더 변환을 보여주는 히어로, 기능 소개 4카드
- **LoginModal**: 구글/카카오 소셜 로그인 + 이메일 회원가입/로그인, 실제 Supabase Auth 연동 완료
- **CalendarPage**: 월간 그리드, 우측 사이드바 자유 텍스트 일정 입력, 날짜 클릭 시 분 단위 드래그 상세 보기,
  반복 일정 "이 날짜만/전체 적용" 토글, 톤온톤 프리셋 5종 + 이미지 레퍼런스 기반 템플릿 4종(모던미니멀/세이지아치/볼드레트로/플레이풀스트라이프),
  칸별 배경색·사진 붙이기, 캘린더 배경 이미지
- **SettingsPage**: 브라우저 푸시 알림(구독과 무관하게 항상 이 방식), 구독 요금제 미리보기, 계정

## 백엔드 (Supabase) — 현재 상태
- **완료**: DB 스키마(`profiles`, `events`, `event_overrides`, `cell_decorations`), RLS 정책, 회원가입 트리거,
  Google/Kakao OAuth 연동, 이메일 인증, 프론트엔드-Supabase CRUD 연동(일정 추가/삭제/수정, 칸 꾸미기, 프로필 저장)
- **미완료**: 아래 "다음 단계" 참고

## 일정 파싱 방식 — 중요, 다음 단계 최우선
현재 `App.jsx`의 `parseClause()` / `tryParseCommand()`는 **정규식 기반 더미 파서**입니다.
"평일", "부터~까지", 쉼표로 나열한 요일, 날짜만 입력 등 자주 쓰는 패턴은 인식하지만,
완전히 자유로운 문장(예: "17일에 회사 안 나가 그 날 일정 지워줘" 같은 부정문·복합 명령)은
못 알아듣는 경우가 있습니다. **실제 서비스로 가려면 이 부분을 AI(LLM) 기반 파싱으로 교체해야 합니다.**
(Claude.ai 채팅에서 이 부분은 다음 단계로 진행 예정이었고, 아직 코드로 안 만들어짐)

## 구독/비용 관련 결정사항
- 메신저는 **카카오톡**으로만 발송 (텍스트 요약 또는 이미지 시간표)
- 알림 기능은 **구독과 무관하게 항상 브라우저 푸시로만** 동작 (카카오톡 알림 아님)
- 카카오 로그인 동의항목: 닉네임(필수), 프로필 사진(선택), 카카오톡 메시지 전송(선택, 구독 기능용),
  친구목록/톡캘린더는 사용 안 함
- 요금제(2026년 7월 기준 견적, **가격은 추후 더 낮게 재조정 예정**):
  무료 0원 / 베이직 3,900원(텍스트 요약 발송) / 프리미엄 6,900원(이미지 시간표 발송, 양식 선택)
- 비용 구조: 카카오톡 이미지 메시지 발송이 구독자당 월 원가에서 가장 큰 비중(약 600원/월),
  AI 파싱 비용은 저가 모델 사용 시 무시할 수준, 결제대행 수수료 약 3%
- 프론트엔드 기본 사용자 이름: "민준님의 캘린더" (더블클릭으로 수정 가능한 더미 기본값, 실제로는 로그인 계정 이름 반영해야 함)
